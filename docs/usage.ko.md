# 사용량 표시 안내

## Codex 계정 토큰 사용량

Codex 계정 사용량은 실험적인 `codex app-server`의
`account/usage/read` 메서드에서 가져옵니다. 응답에는 날짜별 일일 버킷과
누적 합계가 포함되며, 일일 버킷은 해당 날짜가 끝난 뒤 도착할 수 있습니다.

팝오버에는 다음 값을 표시합니다.

- **최근 일일 사용량 (`yyyy-MM-dd`)**: 계정 서비스가 반환한 가장 최신
  날짜의 유효한 버킷 합계입니다. 로컬 날짜가 바뀌어도 계속 표시합니다.
- **이번 달**: 현재 로컬 달력의 이번 달에 속하는 유효한 반환 버킷의
  합계입니다.
- **누적**: 계정 서비스가 반환한 누적 합계입니다.

잘못된 값, 음수 및 미래 날짜 버킷은 제외합니다. 유효한 일일 버킷이 없으면
임의의 값을 만들지 않고 최근 일일 사용량을 사용할 수 없는 상태로 둡니다.
API 환산 비용은 집계 토큰을 이용한 거친 참고값이며 실제 구독 청구액이
아닙니다.

## Claude 메뉴바 쿼터

**설정 → 일반 → 메뉴바**에서 **선택한 제공자 잔여량**을 고르고 제공자를
**Claude Code**로 선택하면 **Claude 쿼터** 선택기가 나타납니다.

- **5시간**: `five-hour` 세션 창
- **일주일**: `seven-day` 계정 창
- **Fable**: 모델 범위의 `scoped-weekly-fable` 창

기존 동작을 유지하기 위해 기본값은 Fable이며 선택은 로컬에 저장됩니다.
현재 계정 응답에 선택한 창이 없으면 다른 쿼터로 바꾸어 표시하지 않고
Claude를 사용 불가 상태로 표시합니다.
